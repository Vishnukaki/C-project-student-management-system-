# Student Management System in C

A simple menu-driven *Student Management System* implemented in C.

## Features
- Add Student
- Display All Students
- Search Student by Roll Number
- Calculate Average Marks

## Project Structure
- src/main.c – C source code
- docs/ProjectReport.pdf – Project report
- docs/ProjectPresentation.pptx – (optional) Presentation
- docs/readme.md – Notes for documents

## How to Run
1. Open src/main.c in any C compiler (CodeBlocks, VS Code, or online compiler).
2. Compile and run.
3. Use the menu to perform operations on student records.
